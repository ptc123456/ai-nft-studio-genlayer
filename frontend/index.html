What is AI NFT Studio?

AI NFT Studio is a dApp built on the **GenLayer Studionet** (Chain ID **61999**). It allows creators to register digital artwork on-chain by running it through decentralized AI jury consensus.

How does the AI Consensus Jury work?

The processing pipeline runs through several asynchronous phases:

  • Submission: The user sends a write transaction with the artwork URL, title, and prompt.
  • Consensus leader: The leader node renders the artwork URL, queries the virtual AI personas (Curator, Skeptic, Ethicist), and aggregates their scores.
  • Validation: Validator nodes run a deterministic check to verify the mathematical correctness and consistency of the leader's scores before committing the block.
  • Settlement: If approved, a GenLayer-native artwork token ID is assigned and stored.
What are the verdict criteria?

Evaluations check Alignment (35%), Quality (25%), Originality (20%), and Safety (20%). If Safety is below 70, the artwork is REJECTED. If Alignment is below 55 or the weighted score is below 70, it is marked as REVISE. Otherwise, it is APPROVED and minted.

What is the Intelligent Contract address?

The contract is deployed on GenLayer Studionet. You can specify the contract address using the VITE_CONTRACT_ADDRESS environment variable in your frontend setup.
